# DAVENGINE-TOOL #

*made by DOUBLE A.V.*

- davengine requires clang and gcc installed for building the game


## What is davengine-tool? ##
davengine tool is a console tool for creating games on game engine davengine.
it already includes davengine and required binaries in release and all you need is to download the release, unzip it, run main.exe, create your game and enjoy game development.

## What is davengine? ##
davengine is a C++ framework 2D game engine that was created by me for developing games on old PCs or laptops.
davengine is based on raylib (C++ framework for creating efficient games)
- system requirements: if your pc is turning on, you can create and run game on davengine. (but now it can only be runned on windows x64 systems)

davengine is super lightweight (8mb - davengine tool + davengine) size will be increasing during the development of davengine
